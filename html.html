HTMLの基礎知識

本章ではHTMLの基本的な構造と知識を学びます。

HTMLにはバージョンがあり、本カリキュラムではHTML5として説明しています。

HTMLとは

  • ウェブ上の文書を記述するための言語
  • Hyper Text Markup Language(ハイパー・テキスト・マークアップ・ランゲージ）の略
  • 文書の構造や意味をhtmlタグとして記述する
  • htmlタグは半角で記述しなければならない
  • html5、xhtml、html4.01などバージョンがある
  • W3Cという団体の勧告で世界的に標準化されている
  • HTML5が現在の主流。
  • index.htmlのように拡張子は.html
参考
  • W3C HTML5(原文)
  • W3C HTML5(日本訳・非公式)
  • W3C準拠のhtmlチェッカー

HTMLソースを確認する

ブラウザ上で右クリックをし、「ページのソースを表示する」をクリックしてみてください。英語の文字列が付与されたページが表示されたと思います。
これがhtmlのルールにしたがって書かれたhtmlファイルに書かれた内容になります。ブラウザはこの内容をルールに沿って人が見やすいように表示してくれています。
それではそのルールを見ていきましょう。

※HTMLソースとはhtmlで書かれた文章を指します。

最も基本的なhtmlの構造

まず以下のソースコードを御覧ください。

      
        <!DOCTYPE html>
    　　<html lang="ja" dir="ltr">
    　　
          <head>
            <meta charset="UTF-8">
            <title>このhtmlのタイトル</title>
            <meta name="description" content="説明文">
          </head>
    
          <body>
            ここに内容を書く
            実際にブラウザ上に表示される部分
          </body>
    
        </html>
      
      ※改行は見やすくするためであり、必須ではありません。

基本的な構造はまずドキュメントタイプを宣言し、
<HTML>～</HTML>でその文書がHTML文書であることを宣言、
<head>～</head>内にヘッダ情報を、
<body>～</body>内にブラウザに表示する内容を記述します。
それでは分解して見ていきましょう。

<!DOCTYPE html>

ドキュメントタイプ=「文書宣言」と呼ばれるものです。HTML5の文書であることを示しています。

<html lang="ja" dir="ltr">

htmlのlang属性は「ja」、dir属性は「ltr」と言う形です。
lang=言語(language)、jaは日本語(japanese)を表しています。
dirは文章の方向を指定する属性でltrはleft to rightつまり左から右の方向へ文章が書かれている事を表します。
右から左(right to left)の場合はrtl(ヘブライ語、アラビア語、ペルシャ語などの場合)とします。
つまり、このhtmlは日本語で左から右へ書かれていると表していることになります。

<head> ～ </head>

文書のタイトル等のヘッダ情報を記述します。
このヘッダ情報はブラウザや検索エンジン向けの記述となり、ブラウザ上で表示されることはありません。

<body> ～ </body>

この中にブラウザ上に表示する記述を行います。

<meta charset="utf-8">

文字コードの指定を表すcharaset属性がutf-8であることを表します。
基本的にutf-8を使うことを推奨します。

<title> ～ </title>

ここに書かれた文字はseoにおいて最も強いと言われています。
検索結果のアンカーテキスト(リンクの文字)になる部分です。
googleやyahoo、bingなど検索結果のタイトルになる可能性が高く、ブラウザのタブにも表示されます。
body内に書かれた内容に沿ったタイトルをページ毎に設定する必要があります。
ページ毎に重複しないユニークなタイトルをつけましょう。
一般的に検索結果で32文字を超えた部分は…となるため、その範囲で書くのが良いとされています。

<meta name="description" content="説明文">

このhtmlファイルのdescription(説明文)をcontent内に書きます。
検索結果の紹介文(スニペット)として表示される可能性が非常に高く、ユーザーが選択肢としてクリックする目安になる重要な部分の一つです。
内容に沿ってユーザーが興味を持つように書くことが大切になります。
PCでは約120文字、スマホでは50文字程度表示されます。
titleタグと同様に他ページと内容が重複しないようにしましょう。

googleでは検索順位には影響しないとしています。
